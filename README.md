# 🚀 Misión Rover: Protocolo de Integración 2026
Bienvenido al sistema de control del Rover Curiosity-II.

## Objetivo
Unificar los tres subsistemas de telemetría en un único script `data_generator.sh` 
y completar el `monitor.sh` para procesar los datos en tiempo real.

## Instrucciones Rápidas
1. Cada ingeniero trabaja en su rama: `feature/sensores`, `feature/motores`, `feature/comms`.
2. Todos deben crear el archivo `data_generator.sh`.
3. Fusionar en `main` y resolver los conflictos de código.

# Opcional
1. Añadimos flags para el control de cada tipo de dato
2. Funcion para generar la media
3. Crear flag para mostrar las medias
